How event publishing will work (for Mike's eyes only, removed in production)

Every event card above will be rendered from a single Google Sheet named GNMBC Events. The church admin opens the sheet, adds a row with the event details, hits Enter. Within seconds, the event appears here on the website. No login to a custom CMS, no rebuilds, no code commits. The sheet is the database, and Google Sheets is a tool they already know how to use.
